# User Management Dashboard

A modern React CRUD application built with React, Vite, Axios, and Tailwind CSS.

## 🚀 Features

* ✅ Add User
* ✅ Edit User
* ✅ Delete User
* ✅ Search Users
* ✅ Filter by Department
* ✅ Sort Users (A–Z / Z–A)
* ✅ Pagination
* ✅ Form Validation
* ✅ Responsive UI
* ✅ Modern Tailwind CSS Design

## 🛠 Tech Stack

* React.js
* Vite
* Tailwind CSS
* Axios
* JSONPlaceholder API
* JavaScript (ES6)

## 📂 Project Structure

```
src/
├── api/
├── components/
├── hooks/
├── utils/
├── App.jsx
└── main.jsx
```

## ⚙️ Installation

```bash
git clone <repository-url>

cd user-management-dashboard

npm install

npm run dev
```
